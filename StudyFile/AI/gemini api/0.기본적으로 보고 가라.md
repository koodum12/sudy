def to_markdown(text):
  return Markdown(textwrap.indent(text,'>',predicate=lambda _:True))
  '''
  text를 줄마다 '>'문자로 들여쓰기 한다.
  이 때 'predicate-lambda _: True'는 모든 줄에 대해 들여쓰기를 적용한다는 조건
  '''
