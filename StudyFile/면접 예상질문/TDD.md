#### TDD(Test-Driven-Development) 
- 테스트 주도 개발을 의미
- 코드를 작성하기 이전에 **테스트 케이스를** 작성하고 테스트를 통과하기 위한 그에 맞는 기능을 작성
- 테스트 중에 실패한 코드에 대해서는 수정해 나아가면서 **반복적인 테스트**를 통한 개발을 하는 작업을 의미

#### TDD 수행 과정
- 테스트 작성
	 ↓
- 실패하는 테스트 실행
	 ↓
- 코드 작성
	 ↓
- 테스트 통과
	 ↓
- 리팩토링
