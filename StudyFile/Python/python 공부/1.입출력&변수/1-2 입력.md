### input
```python
a  = input()
print(a)#입력한 값 출력
```
int 형으로 입력받는 법
```python
a = input(int())
```
float 형으로 입력받는 법
```python
a = input(float())
```

### input 공백으로 구분해서 2개의 변수 받기.
```python
str_list = input().split()#입력 a b str_list안에 값 = [a,b]
print(type(str_list))#class list 출력
print(str_list) #['a','b']
```
