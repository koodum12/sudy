#### ol 추가 설명  && 부가 기능
 - type
	 - A(영문 대문자)
	 - a(영문 소문자)
	 - I(로마 대문자)
	 - i(로마 소문자)
	 - 1(숫자)
	 - type 사용 예
	 - <ol type = "A"> 순서가 대문자 A부터 B C D 이런 식으로 감
