#### alert
알림창 띄우기
ex)
```js
alert("문자열");
```

##### confirm
확인 창 띄우기
ex)
``` js
confirm("문자열");
```

##### prompt
입력 받기
ex)
```js
prompt("띄울 거","기본 값");
```

##### document.write
html로 출력하기
ex)
```js
document.write("<h1>문자열</h1>");
```

##### console.log
콘솔 창에 출력
ex)
```js
console.log("문자열");
```
