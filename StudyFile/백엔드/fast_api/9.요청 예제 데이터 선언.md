#### Pydantic모델 속 추가 JSON 스키마 데이터
생성된 JSON 스키마에 추가될 Pydantic 모델을 위한 `examples`를 선언할 수 있다

python 3.10 버전 기준
```python
from fastapi import FastAPI
from pydantic import BaseModel

app = FastAPI():
	name: str
	description: str | None = None
	price: float
	tax: float | None = None

	model_config = {
	}
```
