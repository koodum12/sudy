응답 모델과 같은 방법으로, 어떤 경로 작동이든 `status_code`매개변수를 사용하여 응답에 대한 HTTP상태 코드를 선언할 수 있다.

- `@app.get()`
- `@app.post()`
- `@app.put()`
- `@app.delete()`
- 기타

```python
from fastapi import FastAPI 
app = FastAPI() 
@app.post("/items/", status_code=201) 
async def create_item(name: str): 
	return {"name": name}
```


`status_code` 매개변수

- 응답에서 해당 상태 코드를 반환.
- 상태 코드를 OpenAPI 스키마(및 사용자 인터페이스)에 문서화.
