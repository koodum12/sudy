### 1.집계 함수란?
여러 행을 그룹화 하여 하나의 값을 반환하는 함수
일반적으로 `GROUP BY`절과 함꼐 사용

##### 1.1 count()
주어진 조건에 맞는 행의 개수를 반환
```sql
SELECT COUNT(*) AS total_rows
FROM 상품;
```

##### 1.2 SUM()
특정 열의 모든 값을 더해서 반환
```sql
SELECT SUM(*) AS total_price
FROM 상품
```


##### 1.3 AVG()
특정 열의 평균 값을 계산
```sql
SELECT AVG(*) AS total_price
FROM 상품
```
